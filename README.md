# Berserk Community

Welcome to the Berserk community hub. This repository hosts discussions, issue tracking, and community resources for Berserk.

## Getting Help

- **[Help & Support](https://github.com/berserkdb/community/discussions/categories/help-support)** — Ask questions and get help from the community
- **[Bug Reports](https://github.com/berserkdb/community/discussions/categories/bug-reports)** — Report bugs for triage
- **[Feature Requests](https://github.com/berserkdb/community/discussions/categories/feature-requests)** — Suggest new features and improvements

## Discussions

- **[Announcements](https://github.com/berserkdb/community/discussions/categories/announcements)** — Team updates, releases, and roadmap
- **[General](https://github.com/berserkdb/community/discussions/categories/general)** — Catch-all conversations
- **[Show & Tell](https://github.com/berserkdb/community/discussions/categories/show-tell)** — Share your integrations, projects, and demos

## Filing Issues

Use the [issue tracker](https://github.com/berserkdb/community/issues/new/choose) to report bugs or request features using our structured templates.

## Security

If you discover a security vulnerability, please follow our [security policy](SECURITY.md). Do not open a public issue.
